# /simulate — Simulador de Incidentes
Aciona o agente incident-simulator para treinar o time com war games realistas.

## Uso
`/simulate`

## O que acontece
1. Pergunta a stack do time e nível de experiência
2. Seleciona um cenário realista (banco, cache, rede, auth, código)
3. Conduz o war game passo a passo
4. Avalia tempo de detecção, diagnóstico e recuperação
5. Gera relatório de maturidade com forças e oportunidades

## Quando usar
- Treinar novos engenheiros on-call
- Preparar o time para incidentes reais
- Avaliar maturidade de resposta a incidentes

## Exemplo
`/simulate`
"Time Java/Spring Boot + PostgreSQL. Nível intermediário. 30 minutos."
