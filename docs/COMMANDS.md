# Commands

## Disponiveis
- `/doctor`: diagnostico do ambiente
- `/forensic`: debug forense
- `/verify`: revisao de codigo IA
- `/simulate`: simulacao de incidentes
- `/navigate`: navegacao web autonoma
- `/perf`: analise de performance
- `/audit`: auditoria de seguranca/dependencias
- `/diagram`: geracao de diagramas
- `/contract-test`: testes de contrato
- `/i18n`: internacionalizacao/localizacao
- `/migrate`: migracao de legado
- `/incident`: postmortem guiado
- `/unbreak`: analise de breaking changes

## Uso
Digite o comando no chat. Exemplo:
```text
/verify
```

## Referencia detalhada
Os detalhes de cada comando estao em `commands/*.md`.
